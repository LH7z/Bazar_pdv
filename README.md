🛍️ Bazar PDV

Sistema de Ponto de Venda (PDV) desenvolvido para gerenciamento de vendas em um bazar físico, com foco em simplicidade, performance e experiência do usuário.

🚀 Sobre o projeto

O Bazar PDV é uma aplicação web que simula um sistema real de caixa utilizado em pequenos comércios.

O sistema permite realizar vendas de forma prática, organizando produtos, carrinho e pedidos em um fluxo intuitivo.

🧠 Objetivo

Este projeto foi desenvolvido com foco em:

Resolver um problema real (controle de vendas de bazar)
Praticar desenvolvimento Full Stack com Rails
Criar um projeto sólido para portfólio profissional
⚙️ Funcionalidades
🛒 Carrinho de compras

📦 Gestão de pedidos
Criação de pedidos a partir do carrinho
Organização das vendas realizadas

💻 Como rodar o projeto
# Clonar repositório
git clone https://github.com/LH7z/Bazar_pdv.git

# Entrar na pasta
cd Bazar_pdv

# Instalar dependências
bundle install

# Criar banco de dados
rails db:create db:migrate db:seed

# Rodar servidor
rails server

Acesse no navegador:

http://localhost:3000
